# Expense Tracker App

A simple **ASP.NET Core MVC** application to track daily expenses.

## Features
- Add, Edit, Delete expenses
- Category-based tracking
- Total expense dashboard
- Clean UI without Bootstrap

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- C#
